# Embed Video Exercise

## Required Files

Place the following files in this folder:

1. **index.html** - The main HTML file with the embedded video
2. **video.mp4** - Your video file (replace with your own video)
3. **poster.jpg** - A poster image for the video

## Requirements Met

- ✅ Video loops continuously
- ✅ Displays a poster image before playback
- ✅ Centered horizontally and vertically on the page
